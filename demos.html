below are some "live" tools you can play with, as well as screencasts showing phylotastic components

  • Topology Server (web form or screencast)

    Want to see a phylogeny right now for a list of species you supply? Click the links above to use a web form or watch the screencast (which shows how to use the form or access the web service via a script).

  • Reconciliotastic (online demo)

    The problem with gene-tree-vs-species-tree reconciliation is that most users don't know where to get a species tree. In this live demo, Reconcili-o-tastic starts with a gene tree, discovers the species for each gene, gets a species tree phylotastically, then runs reconciliation software and displays the results graphically.

  • DateLife (web form or screencast)

    Wouldn't it be great to calibrate your tree using dates of evolutionary divergence estimated by experts, based on fossil evidence (but without the restrictions of TimeTree)? Welcome to the live demo of DateLife!

  • Mesquite-o-tastic (screencast)

    What would it be like to get a species phylogeny without ever leaving your favorite software workbench for evolutionary analysis?

  • Making Galaxy Phylotastic! (screencast)

    Whether you are a Galaxy user or just curious, this screencast will show how to use this popular workflow environment to compose and execute a phylotastic workflow (rectify names, get tree, calibrate using DateLife). Want more? Check out the code and a live server.
